## 🔥 Core Skills

### 🐍 Python
- 파이썬 기본 문법 및 구조 이해
- 함수, 클래스, 모듈화
- 파일 입출력 및 예외 처리

### 📊 Data Analysis & Visualization
- **NumPy**: 수치 연산 및 배열 처리
- **Pandas**: 데이터 전처리 및 분석
- **Matplotlib**: 데이터 시각화 및 분석 결과 표현

### 🤖 AI / Deep Learning
- **TensorFlow / Keras**
  - 신경망 구조 설계
  - 분류(Classification) 및 회귀(Regression) 모델 구현
  - 모델 학습, 평가, 성능 개선 실험

### 👁 Computer Vision
- **OpenCV**
  - 이미지 처리 및 영상 처리
  - 기본 컴퓨터 비전 기법 구현
- **YOLO**
  - 객체 탐지(Object Detection)
  - 모델 실행 및 탐지 결과 시각화
